# LocalAiSearcher 🔍🤖

**LocalAiSearcher** — это полностью автономное десктопное приложение с искусственным интеллектом для работы с документами (Retrieval-Augmented Generation, RAG). 

Оно позволяет загружать ваши документы (PDF, DOCX, TXT, MD) и задавать вопросы по их содержимому. Вся обработка, включая генерацию эмбеддингов и ответов LLM, происходит **исключительно локально** на вашем компьютере.

## 🚀 Ключевые особенности

*   **100% Приватность:** Приложение не требует подключения к интернету ни для установки, ни для работы. Никакие данные не покидают ваше устройство.
*   **Всё включено:** Не нужно дополнительно устанавливать и настраивать другие сервисы. ИИ-модели поставляются в комплекте с инсталлятором.
*   **Кроссплатформенность:** Нативный десктопный интерфейс для Windows, macOS и Linux на базе [Avalonia UI](https://avaloniaui.net/).
*   **Высокая производительность:** Инференс нейросетей работает в едином процессе с приложением благодаря [LLamaSharp](https://github.com/SciSharp/LLamaSharp) и оптимизированному векторному поиску через [sqlite-vec](https://github.com/asg017/sqlite-vec).
*   **Современный стек:** Построено на базе платформы **.NET 10**.

## 🧠 Архитектура и схема работы (RAG)

Процесс работы разделен на два основных потока: **Индексация документов** (Background) и **Ответы на вопросы** (Chat).

```mermaid
flowchart TD
    subgraph Инициализация
        Models[Локальные GGUF модели] -->|LLamaSharp| LLM(LLaMA 3.2 3B)
        Models -->|LLamaSharp| Embed(Nomic Embed Text)
    end

    subgraph Пайплайн документов
        Doc[Документ PDF/DOCX] --> Extractor[Извлечение текста]
        Extractor --> Chunks[Разбиение на чанки]
        Chunks --> Embed
        Embed -->|Векторы 768d| SQLite[(SQLite + sqlite-vec)]
        Chunks -->|Текст чанков| SQLite
    end

    subgraph Чат с ИИ
        User((Пользователь)) -->|Вопрос| Q[Запрос в чат]
        Q --> Embed
        Embed -->|Вектор вопроса| Search{Векторный поиск}
        SQLite --> Search
        Search -->|Топ-5 релевантных чанков| Prompt[Формирование промпта]
        Prompt --> LLM
        LLM -->|Потоковый ответ| User
    end
```

## 🛠 Технологический стек

*   **Платформа:** .NET 10 (C#)
*   **UI:** Avalonia UI (MVVM)
*   **AI Оркестрация:** Microsoft Semantic Kernel
*   **LLM Engine:** LLamaSharp (обертка над llama.cpp)
*   **Векторная БД:** SQLite + расширение sqlite-vec
*   **Парсинг:** UglyToad.PdfPig (PDF), DocumentFormat.OpenXml (DOCX)
*   **Упаковка:** Velopack

## 📦 Модели в комплекте

Приложение по умолчанию использует следующие оптимизированные модели (в формате GGUF):
*   `nomic-embed-text` (~274 MB) — для создания векторных представлений текста.
*   `llama3.2:3b` Q4_K_M (~2.0 GB) — для генерации точных и контекстных ответов.
